# v1.0.1
> 阅读architecture.md后再开始研发

## 小功能修改

1. 修改隐藏图片时，图片应该直接消失，而不是用眼睛符号覆盖，只保留打乱公式。
2. 当开始计时时，刷新、修改公式按钮直接置灰不给用。
3. 刷新公式按钮不能用自定义svg，而是标准的icon库的图标。
4. 需要一个一键收起全部解法（最少步链）
5. 具体解法中，不需要（[]括号） 这个按钮。
6. 手动整理板中的分支，需要做成下拉框，且按照优先级排序。支持过滤。
7. 添加一个github的按钮在左上角，可以跳转到：[https://github.com/guojia99/fmc-paperless](https://github.com/guojia99/fmc-paperless)
8. 添加一个设置弹窗，导入导出放到设置里面。
9. 顶层栏始终维持在最顶部。
10. 键盘有一个最大固定宽度，能够在手机端全部占满，但是平板不能太大，保持目前在底部固定， 居中即可。
